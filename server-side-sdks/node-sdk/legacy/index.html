Node.js Helper

The Plivo Node.js helper simplifies the process of making Plivo API calls and generating Plivo XML. Please note that this is the legacy version.

If you are looking for the latest version, you can get it here.

Prerequisites

  • Node.js
  • NPM

Installation

Quick Installation

Run the following command for a quick installation of the helper.

$ npm install plivo

Manual Installation

Install the helper manually from Github.

$ cd ~/.node_libraries # or the directory where node modules are stored in your OS.
$ git clone git://github.com/plivo/plivo-node.git plivo-node
